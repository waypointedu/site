Verify a Certificate

Each Waypoint certificate carries a unique QR code that points to this page. Enter the certificate ID to confirm authenticity.

Enter the ID exactly as shown on the certificate.

How Verification Works

  • Each certificate includes a unique ID (format: WPT-YYYY-XXXXX)
  • QR codes on certificates link directly to this page with the ID pre-filled
  • Verification confirms course title, completion date, and issuing cohort
  • For privacy, student names are not displayed publicly

Note: This is a demo implementation. In production, verification would query a secure database or static JSON file. For now, the verification is simulated in JavaScript.
